Testando diferentes fundos

Quadrado 1:
Cor sólida de fundo (ivory). Neste quadrado, aprendemos a usar o background-color para aplicar uma cor simples.

Quadrado 2:
Fundo com gradiente. A propriedade linear-gradient cria uma transição suave entre cores.

Quadrado 3:
Fundo com uma imagem. Usamos background-image e background-size: cover para ajustar a imagem ao tamanho do quadrado.
